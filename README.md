# 蹲蹲哇 App 托管页面

本仓库用于 [蹲蹲哇（SquatCounter）](https://github.com/liuv/squatcounter) 的隐私政策、技术支持等公开页面，供 App Store 等平台引用。

## GitHub Pages 配置

1. Settings → Pages
2. Source: **Deploy from a branch**
3. Branch: **main**，Folder: **/ (root)**
4. 保存后访问：https://liuv.github.io/squatcounters-pages/
